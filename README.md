# Data-Analysis-of-German-used-cars-from-eBay
Data Analysis or sometimes referred to as exploratory data analysis (EDA) is one of the core components of data science. 
This repository demonstrates some common exploratory data analysis methods and techniques using python. 


Dataset Overview
	The dataset contains details of the used cars in Germany which are on sale on eBay.
	The dataset is not clean and hence a lot of data cleaning is carried out. For e.g. prices were too high which are replaced by the median and outliers are removed accordingly.
	The dataset is cleaned and stored in a Datapreparation folder which contains the entire cleaned dataset named as cleaned_autos. another folder named cleaned_autos_analysis containing files structures containing subsets of the cleaned dataset.  

Analysis 1 
	This analysis gives the distribution of vehicles based on year of Registration and prices of vehicles based on vehicles types.

Analysis 2
	This analysis gives the number of cars which are available for sale in the entire dataset based on a particular brand.

Analysis 3
	This analysis gives you the average price of the brand of vehicles and their types which are likely to be found in the dataset.
